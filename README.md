# Sales Prediction GUI Application

This repository contains a Python-based GUI application for sales prediction tasks. The application is built using PyQt5 and offers functionality for testing, training, and performance analysis.
Features

•	Testing Interface: Allows users to test predictions.

•	Training Interface: Facilitates the training of the sales prediction model.

•	Performance Analysis: Provides tools to analyze the performance of the trained model.

•	Exit Functionality: A clean exit process from the application.

Installation

To run this application, you need to install PyQt5. You can install the required packages using the following command:

pip install PyQt5

Usage

To start the application, run the MAIN_CODE.py script:

python MAIN_CODE.py
